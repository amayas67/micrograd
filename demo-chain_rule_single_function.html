⛓️ Chain Rule Proof — Single Variable

Deriving the chain rule for $L(f(x))$ using differentials

🎯 Goal

Consider a composite function with a single variable path:

$$L = L\bigl(f(x)\bigr)$$

We want to compute the derivative of $L$ with respect to $x$, that is, $\dfrac{dL}{dx}$.

💡 Simplification: If we had $L(f(x), g(x))$, we could treat $g$ as a constant and focus only on the $f$ path. This gives us the simpler case $L(f(x))$.

1 Differential of $f$

By definition of the derivative, a small change $dx$ in $x$ produces a change in $f$ of:

$$df(x) = f'(x) \cdot dx$$

Therefore, the new value of $f$ is:

$$f(x + dx) = f(x) + f'(x) \cdot dx$$
↳ This is the linear approximation of $f$ near $x$.

2 Differential of $L$ at any point

For any differentiable function $L$, a small change $\varepsilon$ in its input produces:

$$dL = L(u + \varepsilon) - L(u) = L'(u) \cdot \varepsilon \quad \text{as } \varepsilon \to 0$$
↳ Here $u$ is the input of $L$ and $\varepsilon$ is a change in that input — not in $x$. We use $\varepsilon$ to keep it distinct from $dx$.

3 Apply to $L$ evaluated at $u = f(x)$

We now use the formula from Step 2 with $u = f(x)$:

$$dL = L\bigl(f(x) + \varepsilon\bigr) - L\bigl(f(x)\bigr) = L'\bigl(f(x)\bigr) \cdot \varepsilon$$

This is valid for any $\varepsilon \to 0$. The question is now: what is $\varepsilon$ when $x$ changes by $dx$?

↳ $\varepsilon$ is the change in the input of $L$, i.e. the change in $f(x)$. We don't know its value yet — that's what Step 4 computes.

4 Identify $\varepsilon$: the change in $f(x)$

From Step 1, when $x$ changes by $dx$, the change in $f(x)$ is:

$$\varepsilon = f(x + dx) - f(x) = f'(x) \cdot dx$$

Since $f'(x)$ is finite and $dx \to 0$, we have $\varepsilon \to 0$ — so the linear approximation in Step 3 remains valid. We can now substitute $\varepsilon = f'(x) \cdot dx$ into the formula from Step 3:

$$dL = L\bigl(f(x) + \underbrace{f'(x) \cdot dx}_{=\,\varepsilon}\bigr) - L\bigl(f(x)\bigr) = L'\bigl(f(x)\bigr) \cdot \underbrace{f'(x) \cdot dx}_{=\,\varepsilon}$$
🔑 Key step: The substitution $\varepsilon = f'(x) \cdot dx$ is valid because $f'(x)$ is finite, so $\varepsilon \to 0$ as $dx \to 0$ — the linear approximation of $L$ remains exact.

5 Recognize $f(x + dx)$

From Step 1, $f(x) + f'(x) \cdot dx = f(x + dx)$, so the left-hand side is simply:

$$L\bigl(f(x) + f'(x) \cdot dx\bigr) = L\bigl(f(x + dx)\bigr)$$

Substituting back, we obtain the final expression for $dL$:

$$dL = L\bigl(f(x + dx)\bigr) - L\bigl(f(x)\bigr) = L'\bigl(f(x)\bigr) \cdot f'(x) \cdot dx$$
↳ This is the actual change in $L$ when $x$ varies by $dx$.

✅ Conclusion — The Chain Rule

Dividing both sides by $dx$:

$$\boxed{\frac{dL}{dx} = L'\bigl(f(x)\bigr) \cdot f'(x)}$$

Or equivalently, using Leibniz notation:

$$\frac{dL}{dx} = \frac{dL}{df} \cdot \frac{df}{dx}$$

The derivative of a composite function is the product of the derivative of the outer function (evaluated at the inner function) and the derivative of the inner function.

🐍 Connection to micrograd

This chain rule is exactly what each _backward() closure implements in the code. For a node out = self * other, the loss $L$ depends on out, which depends on self and other. The chain rule gives:

$$\frac{dL}{d\texttt{self}} = \frac{dL}{d\texttt{out}} \cdot \frac{d\,\texttt{out}}{d\,\texttt{self}} = \texttt{out.grad} \cdot \texttt{other.data}$$

Which translates directly to:

self.grad  += other.data * out.grad
other.grad += self.data  * out.grad
↳ The += (and not =) accumulates gradients when a node appears in multiple paths of the graph — this is the multivariate chain rule in action.

The topological_order_reversed() function ensures that when _backward() is called on a node, out.grad is already fully computed — so the chain rule application is always valid.

💡 tanh : For out = tanh(self), the local derivative is $1 - \tanh^2(x) = 1 - t^2$, so:
self.grad += out.grad * (1 - t**2)

🔗 Connection to the Multivariate Case

If we had $L(f(x), g(x))$ instead of $L(f(x))$, we would apply this same reasoning to each path separately:

$$\frac{dL}{dx} = \frac{\partial L}{\partial f} \cdot \frac{df}{dx} + \frac{\partial L}{\partial g} \cdot \frac{dg}{dx}$$

Each term is exactly the single-variable chain rule applied to one path, with the other variable held constant.
